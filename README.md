## Things you can do
- Click the `+` button (or hit `Enter`) to add a task.
- Click the red trash icon to delete a task.
- Click the checkmark to mark a task as done.
- Play around with the app.

## Technologies Used
- NextJS
- TailwindCSS
- MongoDB
- Better-Auth
